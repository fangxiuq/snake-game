# 🐍 贪吃蛇游戏

一个使用 HTML、CSS 和 JavaScript 编写的经典贪吃蛇游戏。

## 🎮 游戏特性

- 经典贪吃蛇玩法
- 得分系统和最高分记录（本地存储）
- 精美的视觉效果和动画
- 键盘控制支持

## 🚀 如何运行

直接在浏览器中打开 `index.html` 文件即可开始游戏。

```bash
# 使用 Python 启动本地服务器
cd snake-game
python -m http.server 8080
```

然后访问 http://localhost:8080

## 🎯 操作说明

- **方向键 ↑ ↓ ← →** - 控制蛇的移动
- **空格键** - 暂停/继续游戏
- **开始按钮** - 开始游戏
- **重新开始** - 重置游戏

## 🛠️ 技术栈

- HTML5 Canvas
- CSS3 (渐变、阴影、动画)
- JavaScript (原生)
